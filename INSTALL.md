# Installation

## Prerequisites
- [Python 3](https://www.python.org/)

## Installation

*Windows*
```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

*bash*
```bash
python -m venv venv
source ./venv/bin/activate
pip install -r requirements.txt
```

Issue a server certificate file (default file name is server.pem).
